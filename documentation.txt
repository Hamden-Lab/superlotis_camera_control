how to run this script (C/C++):
server:
'cd /opt/PrincetonInstruments/picam/samples/server-client/bin'
'cd .. && make && cd bin && ./camserver_cit'
cd /opt/PrincetonInstruments/picam/samples/server-client && make && cd bin && ./camserver_cit

client commands from same computer:
in a new terminal window: 
setter: cd /opt/PrincetonInstruments/picam/samples/server-client && echo [cmd]=[arg]| nc localhost 6972
getter: cd /opt/PrincetonInstruments/picam/samples/server-client && echo [cmd]| nc localhost 6972
expose: cd /opt/PrincetonInstruments/picam/samples/server-client && echo expose| nc localhost 6972

client cmds:
exptime [arg: 1000 - 10,000 ms]
exit 
help
shutter_mode [arg: 1, 2, 3]
analog_gain [arg: 1, 2, 3]
temp [arg: -50 - 50 Celcius]
burst
expose
bias
dark


notes: 
shutter_mode: normal = 1, 2 = always_closed, 3 = always_opem
analog_gain: 3 = high, 2 = medium, 2 = low
running a cpp file on its own: 
-g++ -std=c++11 -o [filename] resize_raw.cpp
-g++ -std=c++11 -o [executable] /opt/PrincetonInstruments/picam/samples/server-client/cpp_testing/[filename].cpp -lcfitsio
